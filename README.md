# 📝 LocalNotes

LocalNotes is a simple React-based notes application built as my **first React project**.  
The goal of this project was to understand core React fundamentals and how browser storage works—keeping things simple, functional, and clear.

---

## 🚀 Features

- Write notes using a clean textarea interface
- Save notes locally using a custom key
- Retrieve saved notes from browser `localStorage`
- Controlled inputs and conditional rendering
- No backend, no database — purely client-side

---

## 🛠️ Tech Stack

- **React**
- **JavaScript (ES6)**
- **CSS**
- **Browser localStorage**

---

## 📚 What I Learned

This project helped me understand:

- React `useState` hook
- Handling form submissions
- Controlled form components
- Conditional UI rendering
- Working with `localStorage`
- Basic component structure and state flow
